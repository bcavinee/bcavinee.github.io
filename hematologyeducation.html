This application was designed to provide the laboratory professional with resources to excel in hematology. Performing differentials can be a difficult task due to the variablity of how cells look in different populations. I developed a reference cell function that allows for an organization to build custom cell reference guides based on the patients they encounter everyday. Pictures of cells can be uploaded to the application and saved in a database to be referenced later by the technologist. Hematology Education offers an interactive hematopoiesis function that takes the user through hematopoiesis and provides information for each cell along the way. This application also offers personalized quizzes that can be made from the reference cell database.
